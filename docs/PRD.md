# AI-Powered Code Review & Mentorship Platform

## Overview

The AI-Powered Code Review & Mentorship Platform automatically reviews GitHub Pull Requests using Artificial Intelligence.

Instead of waiting for a senior developer to review code, developers receive instant feedback with explanations, suggested improvements, and best practices.

The platform also tracks a developer's progress over time, helping them improve their coding skills.

---

## Goal

Build a platform that:

* Connects with GitHub
* Detects new Pull Requests
* Reviews changed code automatically
* Posts comments back to GitHub
* Displays review history
* Tracks developer skills

---

## Core Features

* GitHub OAuth Login
* Repository Connection
* GitHub Webhooks
* Background Job Queue
* Static Code Analysis
* AI Code Review
* GitHub PR Comments
* Real-Time Notifications
* Skill Progress Dashboard
